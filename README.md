<>h1>VStreem-X</h1>
<h2>Features</h2>
<li>Users can view  videos</li>
<li>Users can upload videos by login</li>
<li>Users are authenticated and authorized using JWT</li>
<li>Videos are stored in cloudinary</li>
<li>Videos are played in 'react-video'</li>
